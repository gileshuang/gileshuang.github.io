运维向

systemd 是 Linux 下的一款系统和服务管理器，兼容 SysV 和 LSB 的启动脚本。 systemd 的特性有：支持并行化任务；同时采用 socket 式与 D-Bus 总线式激活服务； 按需启动守护进程（daemon）；利用 Linux 的 cgroups 监视进程；支持快照和系统恢复； 维护挂载点和自动挂载点；各服务间基于依赖关系进行精密控制。
